# Blinkit Sales Analysis Dashboard

## Project Overview
This project is a Power BI dashboard developed to analyze Blinkit sales performance across different outlet locations, outlet sizes, item categories, and establishment years.

The dashboard helps stakeholders understand sales trends, customer preferences, and outlet performance to support business decision-making.

---

## Objectives
- Analyze overall sales performance.
- Identify top-performing outlet types and locations.
- Compare sales across outlet sizes.
- Evaluate item category performance.
- Track sales trends over different years.
- Measure customer ratings and average sales.

---

## Key KPIs
- Total Sales: $1.20M
- Average Sales: $141
- Number of Items: 8.5K
- Average Rating: 3.92

---

## Dashboard Features

### 1. Filter Panel
Users can filter data by:
- Outlet Location Type
- Outlet Size
- Item Type

### 2. Sales Trend Analysis
Displays total sales by outlet establishment year to identify growth patterns.

### 3. Fat Content Analysis
Shows sales contribution based on:
- Low Fat Products
- Regular Products

### 4. Item Type Analysis
Analyzes sales distribution across product categories such as:
- Fruits & Vegetables
- Snack Foods
- Dairy
- Frozen Foods
- Household Items
- Others

### 5. Outlet Size Analysis
Compares sales among:
- Small Outlets
- Medium Outlets
- High Outlets

### 6. Outlet Location Analysis
Compares sales performance across:
- Tier 1 Cities
- Tier 2 Cities
- Tier 3 Cities

### 7. Outlet Type Performance
Provides detailed comparison of:
- Total Sales
- Number of Items
- Average Sales
- Average Rating
- Item Visibility

---

## Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV Dataset

---

## Business Insights
- Tier 3 locations generate the highest sales.
- Medium-sized outlets contribute significantly to overall revenue.
- Fruits & Vegetables and Snack Foods are among the top-selling categories.
- Sales performance varies based on outlet type and location.

---

## Outcome
The dashboard provides a comprehensive view of Blinkit's sales performance and helps management make data-driven decisions regarding inventory, outlet expansion, and marketing strategies.
